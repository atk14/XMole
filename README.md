XMole
=====

XMole is a XML parser

Installation
------------

Just use the Composer:

    composer require atk14/xmole

Testing
-------

Install required dependencies for development:

    composer update --dev

Run tests:

    cd test
    ../vendor/bin/run_unit_tests
